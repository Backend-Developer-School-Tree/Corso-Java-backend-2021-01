Il sito internet [Open Library](https://openlibrary.org/) offre agli utenti un archivio su libri liberaremente consultabile mentre a noi developers offre delle [API libere](https://openlibrary.org/developers/api?ref=public-apis) che possiamo interrogare per ottenere informazioni su libri e autori.


Realizzare un software che abbia le seguenti funzionalità:

* Ricerca per nome libro -> Lista di libri
* Ricerca per autore -> Per ogni autore trovato la sua lista di libri
* Ricerca di autore per chiave -> Oggetto autore
* Data la chiave di un libro e quella di un autore, restituire il plot del libro -> Stringa del plot del libro



### Hint

Per creare i querystring in Java, [questa risposta](https://stackoverflow.com/questions/10786042/java-url-encoding-of-query-string-parameters) su Stack overflow può sicuramente esservi d'aiuto
