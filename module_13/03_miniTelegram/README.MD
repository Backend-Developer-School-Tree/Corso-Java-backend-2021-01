Realizzate utilizzando Spark un servizio di messaggistica istantanea che abbia le seguenti funzionalità:

Registrazione nuovo utente
Invio messaggi da utente a utente
Ottieni nuovi messaggi

Il servizio dovrà inviare i congrui messaggi di errore in caso di mal utilizzo delle API

## Hint
Se il servizio di invio messaggi prende in input il nick del mittente e il nick del destinatario allora sarà molto facile impersonarsi un'altra persona, pensate ad un modo per rendere un po' più sicura la piattaforma (o semplicemente, prendete spunto da Telegram, sono abbastanza sicuro che qualcosa per aiutarvi in ciò esiste).
